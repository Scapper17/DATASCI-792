Functional Dependencies Generate Programme

This programme is designed for finding functional dependencies (FDs) between different attributes of a specified database. There are roughly three steps to use. First, the user uploads the db file. Second, experts answer related questions. Finally, the program will output the functional dependencies between the attributes if any.

Instructions

1. Click the "Questionnaire" button to enter the main function page.

2. Follow the instructions of that page to finish two steps, which are uploading a database file, and answer questions with Yes/No.

3. After processing, the result will show on the page.

* Note: If you want to view the pass results, please click the "Results" button on the left.
